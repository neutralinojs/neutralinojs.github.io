Developer Support

If you have questions about Neutralinojs, you can use the following options to find answers.

  • Posting on StackOverflow with the neutralinojs tag.
  • Using the Slack and Discord channels.
  • Mailing to neutralinojs[AT]gmail.com
  • Creating a discussion thread on Github Discussions

Bug reports and feature requests​

There are no perfect software. But we can build good software. Help us to improve Neutralinojs by reporting bugs and submitting feature requests.

Please create an issue on appropriate GitHub repository for feature requests and bug reporting.

  • Neutralinojs native API: Create an issue
  • Neutralinojs JavaScript API: Create an issue
  • neu CLI: Create an issue
