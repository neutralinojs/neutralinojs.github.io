Neutralinojs is a lightweight and portable cross-platform application development framework. It lets you develop native applications using JavaScript/TypeScript, HTML and CSS.

In electron and NWjs you have to install NodeJs and hundreds of dependency libraries. Embedded Chromium and Node creates large overhead and makes even simple apps like “hello world” considerable in size. Neutralinojs offers a lightweight and portable SDK which is an alternative for Electron and NW.js also with many advantages.

If you can build a website, you will be able to build cross-platform desktop apps too because Neutralinojs is a framework for creating native applications with web technologies like JavaScript, HTML, and CSS. Indeed it gives you a set of starter project templates (React, TypeScript, JavaScript, Kotlin and Angularjs) with a full-featured CLI which will make lightweight cross-platform development easier with Neutralinojs.

Neutralinojs Features

Native functions

Neutralinojs Javascript API exposes access to operating system level functions such as file manipulation, executing commands, showing native dialog boxes etc.

Portable & Zero dependency

No extra dependencies are required to run lightweight cross-platforms apps built with Neutralinojs. SDK is fully portable. You will be able to debug applications using your favourite web browser.

Cross-platform support

Neutralinojs apps will work on Linux, Windows and macOS. Single application package can be released for all popular operating systems.

Lightweight & low resource usage

An uncompressed Neutralinojs lightweight cross-platform app is only ~10MB and compressed app size is ~2MB. Indeed, it will not consume either physical memory or storage like any other chromium based cross-platform application development frameworks.
